# Molecule Construction UI (2025-03-18)

Molecule construction UI and preset molecules were added. New scripts: **AtomQuickAddUI** (toolbar with H,C,N,O,S,F,Cl,Br,I,More, Cycloalkanes dropdown, Benzene), **EditModeManager** (edit mode, selection, H-auto, eraser), **MoleculeBuilder** (CreateCycloalkane, CreateBenzene with programmatic bonds), **DisposalZone** (trash zone), **BackgroundClickHandler** (deselect on background click). Modified **AtomFunction**: charge fix (use bondedOrbitals instead of GetComponentsInChildren), GetLoneOrbitalWithOneElectron, ForceInitialize, valence distribution (1-per-orbital), eraser and disposal handling. Modified **CovalentBond**: instant bonds show line immediately. Cycloalkanes and benzene are saturated with hydrogens; cycloalkane H's face outward (90° for C3–C5, 60° for C6); cyclobutane uses a square layout. Bond distance matches manual bonding (1.2×bondRadius).
